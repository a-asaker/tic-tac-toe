# Tic Tac Toe:
Cross-Platform 2-Players Console Tic Tac Toe Game, Coded With Python3.

By : a-asaker


# How To Play:
Just Enter the Position You Want to Put Either `X` Or `Y` In, As It's Displayed On the Game's Graph

                 1-1 | 1-2 | 1-3
                -----+-----+-----
                 2-1 | 2-2 | 2-3
                -----+-----+-----
                 3-1 | 3-2 | 3-3


* Ex :  
       
       * `1-3 or 13`,
       * `3-3 or 33 or 3` And So On ...
